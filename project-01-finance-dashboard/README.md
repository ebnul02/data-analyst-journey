# 📊 Project 01 — Personal Finance Dashboard

## Overview
A personal finance tracker built in Microsoft Excel to analyze 3 months of income and expenses (January–March 2026). The goal was to understand spending habits, calculate monthly savings, and visualize where money is going.

## Tools Used
- Microsoft Excel (Pivot Tables, SUMIF, SUMIFS, Charts)

## What I Did
- Structured raw transaction data across 24 entries with categories, dates, and amounts
- Built a Summary sheet using `SUMIF` and `SUMIFS` formulas to automatically calculate totals
- Tracked key financial metrics: total income, total expenses, net savings, and savings rate
- Broke down spending by category (Rent, Groceries, Transport, Eating Out, Subscriptions, Entertainment)
- Calculated net savings per month to spot trends over time
- Created a bar chart and pie chart to visualize spending distribution

## Key Findings
- Total income over 3 months: **$3,950**
- Total expenses over 3 months: **$3,034**
- Net savings: **$916** (~23% savings rate)
- Rent was the biggest expense at **$2,700** (89% of income goes to rent alone)
- Groceries and Transport were the next largest spending categories

## Skills Learned
- Structuring raw data in a clean, analysis-ready format
- Writing Excel formulas: `SUMIF`, `SUMIFS`, `IF`
- Building dynamic summaries that update when data changes
- Creating charts to communicate financial insights visually
